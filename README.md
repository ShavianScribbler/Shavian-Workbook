# Shavian-Workbook
An introductory workbook for learning to read and write in Shavian.

## What is in Version 1.0
This version just focusses on the basics: incrementally learn the alphabet, practise writing words containing those letters as you progress.

Since it is very introductory, it would probably make sense to add an intermediate/advanced Part 2 to the workbook in a later version. That could include things like longer writing exercises (see Ideas for Version 2.0 below). Also, Shavian letters are intentionally kept separate (i.e. not joined together in the same pen stroke) in this version of the workbook, since in my experience it makes reading and writing more difficult for a newcomer still thinking letter-by-letter and not yet fluent, but this could be added in Part 2.

If you notice any mistakes, please create an issue. If you have any comments, feel free to add them to the Discussions tab.
